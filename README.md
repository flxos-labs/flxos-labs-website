# FlxOS Labs Website

The official website for **FlxOS** — a modular, profile-driven operating system for ESP32 and Desktop platforms.

## Tech Stack
- **HTML5** / **Vanilla CSS**
- **JavaScript** (Modern ES6+)
- **Prism.js** for syntax highlighting
- **Font Awesome** for icons
- **GitHub API** for real-time repository statistics

## Project Structure
- `index.html`: Main landing page
- `about/`: About section
- `docs/`: Documentation section
- `styles.css`: Core design system and global styles
- `script.js`: Main logic, animations, and API integrations

## Recent Optimizations
- **Performance**: Font preloading, lazy loading, and optimized assets.
- **Accessibility**: ARIA labels, semantic HTML, and keyboard navigation.
- **SEO**: Meta tag optimization, dynamic sitemap, and clear heading hierarchy.
- **Features**: Scroll-to-top, latest release fetching, and interactive terminal.

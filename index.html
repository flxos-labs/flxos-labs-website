The Operating System for

A modular, profile-driven OS for the ESP32 family. Built on ESP-IDF and LVGL to deliver rich graphical interfaces and robust performance natively.

View Source Quick Start
flxos-build
$ git clone --recurse-submodules https://github.com/flxos-labs/flxos.git
$ cd flxos
$ python flxos.py select esp32s3-ili9341-xpt
✓ Profile esp32s3-ili9341-xpt selected successfully.
$ python flxos.py build && python flxos.py flash
Building FlxOS... Flashing... Done.

Beyond a bootloader.

FlxOS provides everything needed to build modern appliance interfaces and smart devices in record time.

Rich Graphical Interfaces

Powered by LVGL and LovyanGFX, FlxOS renders gorgeous, hardware-accelerated themes and smooth 60fps animations across almost any display.

Write Once, Run Anywhere

A declarative profile system abstracts hardware. Swap from an ESP32-C3 to an S3 with a single command.

Modular Kernel Architecture

Clean separation of Core, Kernel, UI, and Hardware layers allows for incredible flexibility.

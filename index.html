Ready to Build?

Get your environment set up and start developing with FlxOS in minutes.

Prerequisites

  • ESP-IDF v5.x
  • CMake 3.16+
  • Ninja Build System
Quick Start 
# Clone the repository
git clone --recursive https://github.com/flxos-labs/flxos.git
cd flxos

# Set up environment
. $HOME/esp/esp-idf/export.sh

# Configure and build
idf.py set-target esp32
idf.py build flash monitor

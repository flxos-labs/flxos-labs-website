Advanced Capabilities

Built for performance, scalability, and visual excellence.

Rich GUI & Apps

LVGL-powered touch UI with theming. Built-in apps including Calendar, File Manager, Image Viewer, and Settings.

Multi-Board Support

Profile system for hardware-specific builds supporting ESP32, ESP32-S2, ESP32-S3, ESP32-C3, ESP32-C6, ESP32-H2, and ESP32-P4.

Modular Architecture

Clean separation of Core, Kernel, UI, and Hardware layers for optimal performance and maintainability.

One-Command Builds

Powerful `flxos.py` CLI handles profile selection, target switching, building, flashing, and releasing automatically.

Display-Agnostic

LovyanGFX backend supports ILI9341, ST7789, and many more display controllers, plus a headless mode for embedded server use cases.

Quick Start 
# Clone the repository
git clone --recurse-submodules https://github.com/flxos-labs/flxos.git
cd flxos

# Set up environment
source $IDF_PATH/export.sh

# List and select a profile
python flxos.py list
python flxos.py select esp32s3-ili9341-xpt

# Build and flash
python flxos.py build
python flxos.py flash --port /dev/ttyUSB0
